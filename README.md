### What I learned

Simple media query to adapt padding for smaller screens. Opted to do it under 544px.

```css
@media (max-width: 544px) {
  main {
    padding: 0px var(--spacing-300);
  }

  .card {
    padding: var(--spacing-300);
  }
}
```
